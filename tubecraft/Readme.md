# TubeCraft 服务器

![](../images/TubeCraftTitle.png "TubeCraft 的标题")

> [!warning]
> **TubeCraft 暂时处于开发阶段，很多功能可能还不是很完善，希望你能谅解！**

> [!note] 
> 服务器需要更多有能力的管理，如有意向可加入群私聊。

TubeCraft 是一个 RPG 类型的 **Minecraft Java 版**服务器，其它诸多你曾玩过的 RPG 类型服务器可能有些不同。

## 服务器特色

我们是群组服务器，有生存服/ 创造服/ 小游戏服（计划）。

我们允许红石、建筑。

我们支持正版/ 离线玩家加入。

我们的艺术风格可能更加抽象，玩法可能也少许创新。
