# 加入 TubeCraft 服务器

> [!note]
> 如果你是萌新，强烈建议看完 [萌新的入门课：01 - 下载游戏](noob-tutorial/01-download-game.md) 后在来看这一章节。

## PART 1 添加服务器
点击 `多人游戏` ：
![](../../../images/tutorial/start/choose-server.png "服务器选择界面")

点击 `添加服务器` ：
![](../../../images/tutorial/start/add-server.png)

> [!note]
> 服务器使用了自定义资源包（材质包），请启用 `服务器资源包` 。

服务器地址：
```text
tcbuildmc.xyz
```

## PART 2 加入服务器
点击服务器即可加入。
