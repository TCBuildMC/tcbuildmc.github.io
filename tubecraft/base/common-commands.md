# TubeCraft 服务器常用命令

## TPA 系列

| 命令 | 效果 | 备注 |
| --- | --- | --- |
| `/tpa <玩家名称>` | 向对方发送**把你传送到对方**的传送请求 | - |
| `/tpahere <玩家名称>` | 向对方发送**把对方传送到你**的传送请求 | - |
| `/tpacancel <玩家名称>` | **取消**你发送给对方的传送请求 | 只有**上一个**传送请求能起作用 |
| `/tpaccept` | **接受**对方发来的传送请求 | 同上 |
| `/tpdeny` | **拒绝**对方发来的传送请求 | 同上 |

## Home 系列

| 命令 | 效果 | 备注 |
| --- | --- | --- |
| `/home <家的名称>` | 传送到你所指定的家 | - |
| `/sethome <家的名称>` | 在你脚下设置家 | 每个人至多设置 **10 个**家 |
| `/delhome <家的名称>` | 删除你所指定的家 | - |

## 其他

| 命令 | 效果 | 备注 |
| --- | --- | --- |
| `/warp <地标名称>` | 传送到 TubeCraft 官方设定的传送点位 | - |
| `/spawn` | 返回主城 | - |
| `/rtp` | 随机传送 | 在特殊世界无法使用 |
| `/mail` | 打开邮件箱 | - |
| `/afk` | 挂机 | - |
| `/suicide` | 自杀 | - |
